Инструкция по установке scpVPN для Windows

Скачать scpVPN для Windows

1 Загрузите установочный файл

Нажмите на кнопку выше, чтобы скачать scpVPN. Файл сохранится в папке "Загрузки".

2 Запустите установщик

Найдите файл scpVPN-Windows.exe и запустите его от имени администратора.

3 Завершите установку

Следуйте указаниям мастера установки. Рекомендуем оставить все настройки по умолчанию.
